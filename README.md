# 📊 Dashflow | Sales Analytics (Next.js Edition)

AI-powered sales dashboard built with Next.js, Gemini AI, and Supabase support.

## 🚀 Quick Start

1. **Install Dependencies**:
   ```bash
   npm install
   ```

2. **Configure Environment**:
   Create a `.env` file from the `.env.example` template:
   ```env
   GEMINI_API_KEY=your_key_here
   DATA_SOURCE=csv
   ```

3. **Run Development Server**:
   ```bash
   npm run dev
   ```
   Open [http://localhost:3000](http://localhost:3000) to see the dashboard.

## 📂 Supabase Connection

Want to move beyond CSV files? See our cloud database guide:
👉 **[Supabase Setup Guide](./README_Supabase.md)** (Now integrated into the main logic)

### SQL Table Schema:
```sql
CREATE TABLE sales_data (
  id BIGINT GENERATED BY DEFAULT AS IDENTITY PRIMARY KEY,
  date DATE NOT NULL,
  product TEXT NOT NULL,
  channel TEXT NOT NULL,
  orders INTEGER DEFAULT 0,
  revenue DECIMAL(12,2) DEFAULT 0,
  cost DECIMAL(12,2) DEFAULT 0,
  visitors INTEGER DEFAULT 0,
  customers INTEGER DEFAULT 0
);
```
